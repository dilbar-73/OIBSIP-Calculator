# Simple Calculator 🧮

This is a basic calculator built using *HTML, **CSS, and **JavaScript*. The calculator allows users to perform simple arithmetic operations like addition, subtraction, multiplication, and division. 🧮

## Features
- Performs basic arithmetic operations: +, -, *, / 🧮
- Displays results dynamically 📊
- Clear button (C) to reset the input 🗑
- Handles decimal and negative numbers 🔢

## Technologies Used
- *HTML* for the structure and layout 🌐
- *CSS* for styling the user interface 🎨
- *JavaScript* for the logic behind the calculations 💻
